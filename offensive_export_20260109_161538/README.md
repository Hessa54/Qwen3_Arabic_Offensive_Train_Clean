# Offensive Export Files

This folder contains exported files related to Arabic offensive content
evaluation and analysis using the Qwen3 model.

## Files
- bad_words.json  
  List of curated offensive or prohibited words.

- generate_eval_results_strict.csv  
  Strict evaluation results and metrics.

- prompt_template.txt  
  Prompt template used for model input formatting.

- test_generate_report_strict.json  
  Detailed evaluation report for the test split.

- validation_generate_report_strict.json  
  Detailed evaluation report for the validation split.
